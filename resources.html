Resources on Machine Learning for Hashing

Conferences and Workshops

  • Practical Vector Search Challenge 2023: A competition focused on developing indexing data structures and search algorithms for vector search problems.
  • Billion-Scale Approximate Nearest Neighbor Search Challenge: NeurIPS’21 Competition Track: A large-scale challenge to improve approximate nearest neighbor search methods.
  • Compact and Efficient Feature Representation and Learning in Computer Vision, ICCV 2019: A workshop focused on compact feature learning, including binary hashing methods.
  • International Conference on Similarity Search and Applications: A conference dedicated to similarity search techniques and applications.
  • Joint Workshop on Efficient Deep Learning in Computer Vision: Co-located with CVPR 2020, focusing on efficient deep learning techniques.

Hashing Software Packages

  • Faiss (Facebook AI Similarity Search): A powerful library for efficient similarity search and clustering of dense vectors, used for large-scale datasets.
  • Annoy (Approximate Nearest Neighbors Oh Yeah): Developed by Spotify, this C++ library is designed for fast approximate nearest neighbor search in high-dimensional spaces.
  • Deep Hashing Toolbox

Books

Here are a few recommended books on large-scale machine learning:

  • Mining of Massive Datasets: Covers large-scale data mining topics, including a detailed section on LSH.
  • Introduction to Information Retrieval: A classic textbook on information retrieval, featuring comprehensive coverage of data indexing and retrieval techniques.
  • Efficient Processing of Deep Neural Networks: A detailed exploration of techniques for processing deep neural networks efficiently.
