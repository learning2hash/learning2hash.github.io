Using hashing techniques, this paper develops a family of space-efficient Las Vegas randomized algorithms for \\(k\\)-SUM problems. This family includes an algorithm that can solve 3-SUM in \\(O(n^2)\\) time and \\(O(\sqrt&#123;n&#125;)\\) space. It also establishes a new time-space upper bound for SUBSET-SUM, which can be solved by a Las Vegas algorithm in \\(O^*(2^&#123;(1-\sqrt&#123;\8/9\beta&#125;)n&#125;)\\) time and \\(O^*(2^&#123;\beta n&#125;)\\) space, for any \\(\beta \in [0, \9/32]\\).
