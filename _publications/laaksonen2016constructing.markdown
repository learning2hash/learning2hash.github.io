<p>Let <span class="math inline">\(A_2(n,d)\)</span> be the maximum size
of a binary code of length <span class="math inline">\(n\)</span> and
minimum distance <span class="math inline">\(d\)</span>. In this paper
we present the following new lower bounds: <span
class="math inline">\(A_2(18,4) \ge 5632\)</span>, <span
class="math inline">\(A_2(21,4) \ge 40960\)</span>, <span
class="math inline">\(A_2(22,4) \ge 81920\)</span>, <span
class="math inline">\(A_2(23,4)
\ge 163840\)</span>, <span class="math inline">\(A_2(24,4) \ge
327680\)</span>, <span class="math inline">\(A_2(24,10) \ge
136\)</span>, and <span class="math inline">\(A_2(25,6) \ge
17920\)</span>. The new lower bounds are a result of a systematic
computer search over transitive permutation groups.</p>
