Beating Fredman-komlos For Perfect k-hashing

Guruswami Venkatesan, Riazanov Andrii. Arxiv 2018

[Paper]    
ARXIV Independent

We say a subset {% raw %}\(C \subseteq \{1,2,\dots,k\}^n\){% endraw %} is a {% raw %}\(k\){% endraw %}-hash code (also called {% raw %}\(k\){% endraw %}-separated) if for every subset of {% raw %}\(k\){% endraw %} codewords from {% raw %}\(C\){% endraw %}, there exists a coordinate where all these codewords have distinct values. Understanding the largest possible rate (in bits), defined as {% raw %}\((\log_2 C )/n\){% endraw %}, of a {% raw %}\(k\){% endraw %}-hash code is a classical problem. It arises in two equivalent contexts: (i) the smallest size possible for a perfect hash family that maps a universe of {% raw %}\(N\){% endraw %} elements into {% raw %}\(\{1,2,\dots,k\}\){% endraw %}, and (ii) the zero-error capacity for decoding with lists of size less than {% raw %}\(k\){% endraw %} for a certain combinatorial channel. A general upper bound of {% raw %}\(k!/k^{k-1}\){% endraw %} on the rate of a {% raw %}\(k\){% endraw %}-hash code (in the limit of large {% raw %}\(n\){% endraw %}) was obtained by Fredman and Koml'{o}s in 1984 for any $k \geq 4{% raw %}\(. While better bounds have been obtained for \){% endraw %}k=4$, their original bound has remained the best known for each {% raw %}\(k \ge 5\){% endraw %}. In this work, we obtain the first improvement to the Fredman-Koml'{o}s bound for every {% raw %}\(k \ge 5\){% endraw %}. While we get explicit (numerical) bounds for {% raw %}\(k=5,6\){% endraw %}, for larger {% raw %}\(k\){% endraw %} we only show that the FK bound can be improved by a positive, but unspecified, amount. Under a conjecture on the optimum value of a certain polynomial optimization problem over the simplex, our methods allow an effective bound to be computed for every {% raw %}\(k\){% endraw %}.
